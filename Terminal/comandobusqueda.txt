Para realizar la busqueda de todos los archivos txt en la ruta de home usamos el siguiente comando
find ./ -type f -name '*.txt'
Si deseamos agregar dicha busqueda a un archivo de texto utilizamos el simbolo de rediccionamiento
find ./ -type f -name '*.txt' > mytextfiles.txt
Si ademas queremos lanzar un mensaje que no avise cuando se halla guardado el archivo utilizamos cowsay o echo
find ./ -type f -name '*.txt' > mytextfiles.txt 2>&1 cowsay 'Archivos guardados exitosamente!!!!'
Debemos agregar la orden 2>&1 para derivar tanto la salida del output como de los errores al archivo de texto
Posteriormente cuando deseemos visualizar el contenido del archivo de texto podemos agregar ademas el comando
lolcat con el cual veremos el contenido en colores.
cat mytextfiles.txt | lolcat
